Network Security project in Microsoft Azure

This case study required me to deploy a virtual network to enhance Network Security in Microsoft Azure. I successfully deployed Vnet, VMs, Fortinet Firewall and Microsoft Sentinel in cloud. All the traffic from VMs to WAN and from WAN to VMs pass through Firewall for security. At the end of this project, I was able to import IPS (Intrusion Prevention System) signature in Fortinet Firewall to detect and block Brute Force attacks on RDP (Remote Desktop Protocol) and create custom alerts on Microsoft Sentinel.

Network design for a Hospital

In order to improve security between two branches of hospital, I implemented network design while considering importance of CIA triad (Confidentiality, Integrity, Availabilitiy). I configured LAN and WAN, Wireless connectivity for end users, Redundancy to avoid single point of failure, Network Security with IPsec VPN Encyrption and DHCP server to allocate IP addresses dynamically.

Network Infrastructure for a Company

This case study involved migration of a company's infrastructure which required planning for LAN distribution, IP subnetting, Use of public IP addresses to internet and DHCP server. I successfully desgined a network where VLANS could interconnect with each other and configured NAT (Network Address Translation) to convert Private IP addresses to Public IP address. DHCP is also configured to assign IP addresses dynamically. This design is sufficient for the staff of upto 600 people.

Bank network design for redundancy

In order to provide redundancy in Bank, I implemented a hierarchical model (Access layer, Distribution layer, Core layer) where IT staff is able to manage the entire Network remotely with SSH. OSPF Routing protocol is enabled to advertise routes. Port security is also enabled on switches for security.
